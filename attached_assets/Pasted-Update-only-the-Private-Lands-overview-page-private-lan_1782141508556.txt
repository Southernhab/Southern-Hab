Update only the Private Lands overview page:

/private-land-services/index.html

Use this image as the full-page service-section background:

/images/private-lands-management.png

DESIGN GOAL

Create a strong Private Lands services section with the service cards floating over the landscape image.

BACKGROUND

Apply the image to the main Private Lands services area.

Use:

- background-image: url('/images/private-lands-management.png')
- background-size: cover
- background-position: center
- background-repeat: no-repeat
- background-attachment: scroll

Do not use a gradient.

Add a solid semi-transparent forest-green overlay above the image so the section remains readable.

Preferred overlay:

background-color: rgba(19, 39, 31, 0.70);

Use either:

- a pseudo-element overlay
- or a separate absolutely positioned overlay layer

Do not apply opacity to the entire section because that would also fade the cards and text.

SECTION STRUCTURE

The order should be:

1. Private Lands heading
2. Short overview text
3. Service-card grid
4. Private Land Inquiry button

Keep the existing approved wording and links unchanged.

SERVICE CARDS

Place all Private Lands service cards over the background image.

Use:

- warm-white card background: rgba(250, 249, 245, 0.96)
- dark text: #252A27
- border: 1px solid rgba(255,255,255,0.35)
- subtle shadow
- consistent rounded corners
- enough internal padding
- equal visual height where practical

Use this visual treatment:

.service-card {
  background: rgba(250, 249, 245, 0.96);
  color: #252A27;
  border: 1px solid rgba(255,255,255,0.35);
  box-shadow: 0 12px 30px rgba(0,0,0,0.18);
  backdrop-filter: blur(2px);
}

Keep card titles, descriptions, and buttons fully readable.

DESKTOP LAYOUT

Above 768px:

- Use a responsive two- or three-column service-card grid
- Keep consistent gaps between cards
- Center the grid within the page
- Do not stretch cards excessively wide
- Allow the image to remain visible between cards
- Keep the heading and introduction in a readable-width text block
- Use cream or warm-white text above the cards

MOBILE LAYOUT

At 768px and below:

- Use one card per row
- Use 16px to 18px horizontal padding
- Make cards full-width
- Make service buttons full-width
- Keep background-position: center
- Change background-attachment to scroll
- Increase the overlay opacity slightly if needed for text readability
- Keep card descriptions limited to 2–3 lines
- Do not allow headings to break in the middle of words
- Do not create horizontal scrolling

Use:

@media (max-width: 768px) {
  .private-services-grid {
    grid-template-columns: 1fr !important;
  }

  .private-lands-services-section {
    background-position: center;
    background-attachment: scroll;
  }
}

ACCESSIBILITY

- Keep text contrast at WCAG AA
- Add descriptive alt text only if the image is also inserted as an img element
- If it is used only as a CSS background, do not add an empty decorative image element
- Keep all cards and links keyboard accessible
- Preserve visible focus states

DO NOT

- add the image to individual service pages
- change service names
- change service links
- change approved copy
- add gradients
- move or change forms
- change navigation
- change the footer
- run screenshot verification
- analyze screenshots
- start browser automation
- run appearance-check loops

Save and stop immediately after applying the Private Lands background and floating-card design.
